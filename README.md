### What is this repo?

- ASP.NET Core
- VueJS 2.*
  - Server-side rendering

### Getting Started?

- Fork and Clone repo
- `yarn install && webpack`
- Hit `F5` in VSCode or Visual Studio 
- Enjoy :)

### Extras

- Get Chrome DevTools for Vue [here](https://chrome.google.com/webstore/detail/vuejs-devtools/nhdogjmejiglipccpnnnanhbledajbpd)
